# work_wx（企业微信）
开发不易维护不易,希望对你开发有帮助！


项目框架springboot+mongodb

该项目包括企业微信所有接口

重点功能外部联系人管理，通讯录同步，会话存档记录文件同步，以及企业微信账号授权登录。

# 注意事项
很多朋友在解密的时候出现解密失败,java项目需要把pkcs1转成pkcs8!
